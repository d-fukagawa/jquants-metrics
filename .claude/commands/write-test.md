# テストコード作成

Vitest を使って、このプロジェクトの既存テスト構成に合わせたテストを作成してください。

## 参照すべきファイル

@src/services/financialService.test.ts
@src/services/stockService.test.ts
@src/jquants/client.test.ts

## 作成手順

1. 対象ファイルと同じディレクトリに `*.test.ts`（または `*.test.tsx`）を作成する
2. 既存テストのスタイル（`describe`, `it`, `expect`）に倣う
3. サービス関数は純粋関数なのでモック不要でテスト可能
4. DB を使うテストは必要に応じてモック化する
5. テスト作成後に `npm run test` を実行して全テストが通ることを確認する

## テスト方針

- エッジケース（null / 0 除算 / 空配列）を必ずカバーする
- 関数の正常系・異常系の両方をテストする
- テスト名は日本語で書いても OK（可読性優先）
