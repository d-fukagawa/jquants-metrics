# バグ調査・修正

このプロジェクトのバグを調査して修正してください。

## 調査手順

1. エラーメッセージ・症状から関連ファイルを特定する
2. `src/routes/` → `src/services/` → `src/db/` の順で原因を追う
3. Cloudflare Workers 固有の問題か確認する
   - `process.env` を使っていないか
   - Node.js 専用 API（`fs`, `path`）を使っていないか
4. JQuants API v2 の認証（`x-api-key` ヘッダー）が正しいか確認する
5. Drizzle の `numeric` 型を `Number()` 変換なしに計算していないか確認する

## 修正手順

1. 根本原因を特定する（症状の隠蔽は禁止）
2. 修正を実施する
3. 修正箇所をカバーするテストを作成または既存テストを更新する
4. `npm run test` を実行して全テストが通ることを確認する
